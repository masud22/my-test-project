# Modern UI Test Project

A simple but beautiful glassmorphism web interface featuring a live animated analog watch face, a registration page, and a login page.

## 🌟 Features

- **Analog Watch Face & World Clock**: A sleek, real-time analog clock and secondary digital world clocks for NY, London, and Tokyo.
- **Glassmorphism Design**: Frosted glass panels, subtle borders, and smooth gradients.
- **Authentication Pages**: Beautiful UI templates for both Login (`login.html`) and Registration (`register.html`) that match the main theme.
- **Daraz E-commerce Clone**: A newly added, full-width e-commerce storefront layout (`daraz.html`) styled exactly like the popular Daraz app.
- **Responsive Layout**: Designed to look great and stay centered on any screen size.

## 🚀 How to Run locally

Since these are static HTML/CSS/JS files, you can simply open the `index.html` file in any supported modern web browser!

If you are using the Vercel CLI locally, you can start a development server with:

```bash
vercel dev
```

Then visit `http://localhost:3000` in your web browser.
